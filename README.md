## Vision
Generative video is the first internet workload that **cannot scale centrally**.  
AFASTGRok MESH solves it by transforming the world's dormant storage into a living, breathing, encrypted mesh dedicated to AI-generated video chunks.

- No more 60-second renders → **<1 second** delivery via edge cache  
- ISP routers become exabyte-scale CDNs overnight  
- Recycled phones in India/Africa earn $MESH rewards  
- End-to-end encrypted — zero-knowledge for all users

## Tech Stack
- **Core**: Rust + libp2p + QUIC/WebRTC
- **Encryption**: XChaCha20-Poly1305 (per-user keys)
- **Storage Proofs**: Verifiable Delay Functions (VDF)
- **Consensus**: Proof-of-Storage + Proof-of-Bandwidth
- **License**: Apache 2.0
